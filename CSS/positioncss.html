CSS Position Property Examples

Position: static

This is the default position. Elements flow normally.

Position: relative

This box is offset from its normal position (top:10px, left:20px).

Position: absolute

This box is positioned absolutely inside the relative container.

Position: sticky

This box sticks to the top when scrolling.
