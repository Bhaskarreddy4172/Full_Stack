CSS Grid Property

grid: "header header header header 100px sidebar content content content 100px footer footer footer footer 100px" ; There are three rows and four columns. The header is first row and takes four columns, sidebar and content are second row and take single column and 3 columns respectively and footer is third column and takes all four columns. All rows have 100px height.
